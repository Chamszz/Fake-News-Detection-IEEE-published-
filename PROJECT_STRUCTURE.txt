FILE DESCRIPTIONS
=================

src/ folder (Source code)
------------------------
preprocessor.py        - Handles NLP preprocessing
                        Functions: lowercase, clean, tokenize, normalize

bert_embedder.py       - BERT feature extraction
                        Functions: load model, generate embeddings (768-dim)

xgboost_classifier.py  - XGBoost classifier implementation
                        Functions: train, predict, evaluate, save model

data_loader.py         - Load and prepare datasets
                        Functions: load CSV, combine sources, split train/test

evaluator.py           - Calculate performance metrics
                        Functions: accuracy, precision, recall, F1, ROC-AUC

utils.py               - Helper functions
                        Functions: logging, file handling, config management


notebooks/ folder (Jupyter Notebooks)
-------------------------------------
01_EDA.ipynb           - Exploratory Data Analysis
                        Content: data distribution, label balance, visualizations

02_Preprocessing.ipynb - Text preprocessing walkthrough
                        Content: cleaning steps, tokenization examples

03_BERT_Features.ipynb - Generate BERT embeddings
                        Content: embedding generation, dimensionality

04_Model_Training.ipynb - Train models
                        Content: training loop, cross-validation, hyperparameters

05_Results_Analysis.ipynb - Results visualization
                        Content: plots, comparisons, performance metrics


data/ folder (Datasets)
-----------------------
raw/                   - Original datasets (not modified)
                        From: Kaggle, ISOT, PolitiFact, Snopes

processed/             - Cleaned and split data ready for modeling
                        Format: CSV with columns (title, text, label)
                        Label: 0=fake, 1=real


models/ folder (Trained Models)
-------------------------------
bert_embedder.pkl      - Trained BERT model
                        Size: ~400MB (not committed to git)
                        Auto-generated during training

xgboost_classifier.pkl - Trained XGBoost classifier
                        Size: Small (~1MB)
                        Can be committed to git

preprocessor.pkl       - Fitted preprocessor object
                        Contains tokenizer and normalization rules


results/ folder (Output Files)
------------------------------
confusion_matrix.png   - Model performance matrix
                        Shows TP, TN, FP, FN values

roc_curve.png          - ROC curve with AUC score
                        Shows sensitivity vs specificity tradeoff

performance_metrics.json - All performance numbers
                        Accuracy, Precision, Recall, F1, ROC-AUC

model_comparison.csv   - Compare different models
                        Rows: each model, Cols: metrics

GIT STRUCTURE
=============

When the script runs, it will:

1. Initialize git in your folder
2. Add all files in logical batches
3. Create commits with appropriate dates
4. Push to GitHub

Files included in commits:
- .py files (source code)
- .ipynb files (notebooks)
- .md files (documentation)
- .txt files (text docs)
- .csv files (processed data)
- requirements.txt
- .gitignore

Files NOT included in commits:
- .git folder (git internal)
- __pycache__ (Python cache)
- *.pkl files over 90MB
- *.pth files (model weights)
- Raw datasets (over 100MB)

These exclusions are in .gitignore
